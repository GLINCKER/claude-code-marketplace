## How to Update on GitHub

### Update Description:
1. Go to your repository: `https://github.com/GLINCKER/claude-code-marketplace`
2. Click the gear icon ⚙️ next to "About" (top right of the page)
3. Paste the repository description above
4. Click "Save changes"

### Update Topics:
1. In the same "About" settings dialog
2. Add topics one by one from the list above
3. GitHub will auto-suggest as you type
4. Click "Save changes"

### Update README Social Preview (Optional):
1. Go to Settings → Options → Social preview
2. Upload a banner image (1200×630px recommended)
3. This will show when the repo is shared on social media

## Recommended Settings

### General Settings:
- **Allow** Issues (for bug reports and feature requests)
- **Allow** Discussions (for community Q&A)
- **Allow** Wiki (for extended documentation)
- **Require** PR reviews before merging
- **Enable** automatically delete head branches

### Branch Protection (main):
- Require pull request reviews before merging (at least 1 approval)
- Require status checks to pass before merging
- Require branches to be up to date before merging
- Include administrators

### Community Health Files:
